DP Sync version 5.1.61
is now available!
This is a large update that includes a number of
fixes and new features
  • You can now create specifications in DP from your Revit projects using
    Quickstubs. This allows you to create full specifications directly out of your
    Revit project without having to create them first in DP
  • Project Phasing has been added, so you can now sync items from specific
    project phases
  • Supports Revit versions 2016-2022
Download
This version requires a fresh install of DP Sync and will not update
automatically in Revit like previous versions.
